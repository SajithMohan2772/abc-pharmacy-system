# ABC-PHARMACY-SYSTEM

This zipped file folder contains a frontend built with React and a backend built with Golang.

## Frontend Setup (React)

### Prerequisites
- Node.js and npm should be installed.

### Installation
1. Unzip the folder to your desired location.
2. Navigate to the frontend directory within the unzipped folder.
3. Open a terminal or command prompt in this directory.

### Usage
4. Install dependencies: Run `npm install` in the terminal.
5. To start the development server: Run `npm start`.
6. Access the application at: `http://localhost:3000`.

## Backend Setup (Golang)

### Prerequisites
- Golang should be installed.

### Installation
1. Unzip the folder to your desired location.
2. Navigate to the backend directory within the unzipped folder.
3. Open a terminal or command prompt in this directory.

### Usage
4. Set up Go modules: Run `go mod init github.com/your-username/project-name` in the terminal.
5. Install dependencies: Run `go mod tidy`.
6. To run the backend server: Run `go run main.go`.
7. The backend will run at: `http://localhost:8080`.

## Note
- Update "./middleware/handler.go" connection string (environment variables) or configurations before running the servers.
